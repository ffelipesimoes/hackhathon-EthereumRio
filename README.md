# universiDAO

A 1º organização autômona e descentralizada (DAO) estudantil da América Latina.
Temos como missão facilitar o levantamento de fundos para projetos de estudantes universitários que estão buscando impactar positivamente e ganhar experiência em web3.

## Running the code

### **FrontEnd**

`git clone https://github.com/ffelipesimoes/hackhathon-EthereumRio.git universiDAO`

`cd universiDAO`

`yarn` - Para instalar as dependências.

`yarn start` - Para instalar as dependências.

### **BackEnd**

Este projeto demonstra um caso de uso básico do Hardhat. Ele vem com um contrato de amostra, um teste para esse contrato, um script de amostra que implanta esse contrato e um exemplo de implementação de tarefa, que simplesmente lista as contas disponíveis.

Tente executar algumas das seguintes tarefas:

```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```

## **Problema**

Apesar da revolução web3 estar acontecendo em frente aos nossos olhos, a universidade não está preparando os estudantes para aproveitar as oportunidades fornecidas pelo mercado.

Ou seja, para muitos estudantes a universidade está defasada e novos atores são necessários neste ecossistema.

No Brasil, há milhões de pessoas matriculadas no ensino superior. Destas, muitas querem aproveitar as oportunidades criadas por empresas web3 mas não sabem como dar os primeiros passos neste novo mundo.

## **Solução**

Assim, como universitários que somos e percebendo esta dor em nossa comunidade, fundamos a 1° organização autônoma e descentralizada estudantil na América Latina. Temos como missão facilitar o fundraising para projetos de estudantes universitários interessados em impactar positivamente e ganhar experiência na área de web3.

A UniversiDAO fomenta que o estudante proponha projetos de pesquisa, desenvolvimento, negócios ou outras práticas podendo receber fundos caso tais projetos sejam aprovados por nossa comunidade.

### universiDAO na prática

A universiDAO tem sua porta de entrada no Discord: https://discord.gg/sVTQbA7q. Aqui é feito a validação do seu vínculo estudantil para fazer parte da DAO.

Uma vez parte da universiDAO, como estudante você poderá requisitar investimentos para fomentar projetos de web3 por meio de uma requisição de fundos que será votado pela universiDAO.

A votação das propostas acontecem na universiDAO e a liberação do fundo acontece logo após o fim da votação.
